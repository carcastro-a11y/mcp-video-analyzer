# Hip Position (bad) — Breaststroke

Frame samples illustrating poor hip position during breaststroke. Hips are dropped too low in the water, increasing drag and reducing forward propulsion through the kick and glide phases.

## Frames

Fourteen frames captured chronologically from a swim video, showing the hip-position issue across multiple stroke cycles from both surface and underwater angles.

`frame_01.png` through `frame_14.png` — ordered by timestamp.

## Purpose

Reference set for the swim-analysis skill. Demonstrates the "bad hip position" failure mode that the analyzer should flag when reviewing competitive breaststroke footage.
