# Hip Position (good) — Breaststroke

Frame samples illustrating proper hip position during breaststroke. Hips stay high in the water and aligned with the body line, minimizing drag and maximizing forward propulsion through the kick and glide phases.

## Frames

Thirteen frames captured chronologically from a swim video, showing correct hip position across multiple stroke cycles from both surface and underwater angles.

`frame_01.png` through `frame_13.png` — ordered by timestamp.

## Purpose

Reference set for the swim-analysis skill. Demonstrates the "good hip position" benchmark — paired with `hip-position-bad/` to give the analyzer a contrastive comparison set when reviewing competitive breaststroke footage.
