# Head Position (bad) — Breaststroke

Frame samples illustrating poor head position during breaststroke. Common faults include the head riding too high during the breath, lifting too early before the pull is complete, or staying lifted too long — all of which break body alignment and increase drag.

## Frames

Twelve frames captured chronologically from a swim video, showing head-position issues across multiple stroke cycles from both surface and underwater angles.

`frame_01.png` through `frame_12.png` — ordered by timestamp.

## Purpose

Reference set for the swim-analysis skill. Demonstrates the "bad head position" failure mode that the analyzer should flag when reviewing competitive breaststroke footage.
