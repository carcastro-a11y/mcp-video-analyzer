# Catch (bad) — Breaststroke

Frame samples illustrating a poor catch phase during breaststroke. Common faults include dropping the elbows below the wrists ("dropped elbow"), letting the hands slip outward without engaging the water, or starting the pull before the arms are fully extended — all of which waste the propulsive potential of the catch.

## Frames

Fifteen frames captured from swim video, showing catch-phase issues across multiple stroke cycles from both surface and underwater angles. Frames 13–15 add additional reference samples.

`frame_01.png` through `frame_15.png` — ordered by timestamp.

## Purpose

Reference set for the swim-analysis skill. Demonstrates the "bad catch" failure mode that the analyzer should flag when reviewing competitive breaststroke footage.
