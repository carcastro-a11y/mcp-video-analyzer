# Catch (good) — Breaststroke

Frame samples illustrating a strong catch phase during breaststroke. The hands stay slightly above the elbows ("high elbow"), the forearms engage as paddles, and the swimmer holds water cleanly through the outsweep before initiating the pull.

## Frames

Thirteen frames captured chronologically from a swim video, showing correct catch mechanics across multiple stroke cycles from both surface and underwater angles.

`frame_01.png` through `frame_13.png` — ordered by timestamp.

## Purpose

Reference set for the swim-analysis skill. Demonstrates the "good catch" benchmark — paired with `catch-bad/` to give the analyzer a contrastive comparison set when reviewing competitive breaststroke footage.
