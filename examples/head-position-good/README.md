# Head Position (good) — Breaststroke

Frame samples illustrating proper head position during breaststroke. The head stays neutral and aligned with the spine, lifts only enough to clear the water for the breath, and returns down quickly so the body can extend through the glide phase.

## Frames

Thirteen frames captured chronologically from a swim video, showing correct head position across multiple stroke cycles from both surface and underwater angles.

`frame_01.png` through `frame_13.png` — ordered by timestamp.

## Purpose

Reference set for the swim-analysis skill. Demonstrates the "good head position" benchmark — paired with `head-position-bad/` to give the analyzer a contrastive comparison set when reviewing competitive breaststroke footage.
